Install memnant.

macOS / Linux
$ curl -fsSL memnant.com/install.sh | sh copy

Detects your OS and architecture. Installs to ~/.memnant/bin.

Windows
> irm memnant.com/install.ps1 | iex copy

PowerShell. Installs to %USERPROFILE%\.memnant\bin.

npm
$ npx memnant copy

Requires Node.js 20+. Works without installing globally.

Then

First run creates your config, ledger, and registers the MCP server for your agent. After that, just:

$ memnant
